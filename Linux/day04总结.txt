一：设置权限与所属用户（组）
    1. chmod修改文件或目录的权限
       chmod [选项] [用户类型] [+|-|=] [权限字符]  文件或目录
       选项： -R 目录与其包含的文件和子目录都设置权限

       用户类型：u  文件的拥有者
                 g   文件的所属组
                 o   其他用户
                 a   所有用户

       权限字符：r 读取权限
                 w 写入权限
                 x  执行权限
         注意：对于目录，r权限代表可以浏览(ls)目录中包含的内容
                        w权限代表可以在该目录下创建、删除、移动文件或子目录
                        x权限代表可以进入(cd)该目录

       补充：chmod还可以使用八进制数设置权限
         eg: chmod 666 apple.txt
             分析：一个八进制数6可以拆分成三个二进制数：110。1代表有相应权限，0
                   代表没有相应权限。

    
     2. chown更改文件（或目录）的拥有者和所属组
        chown [选项] 拥有者:所属组  文件（或目录）
          选项：-R 目录与其包含的文件和子目录都设置所属用户和所属组
      
          eg: chown tianpan:mayor apple.txt # 更改apple.txt的所属用户和组
              chown -R alice:tianpan test2 # 更改test2目录及其内部文件的所属用户和组

          注意：如果只想更改文件所属用户: chown 拥有者 文件(或目录)
              如果只想更改文件的所属组： chown :所属组 文件(或目录)

     3. chgrp改变文件(或目录)的所属组
        chgrp [选项] 更改后的所属组名称 文件（或目录）
          选项：-R 目录与其包含的文件和子目录都设置所属组

     补充：newgrp用来切换有效组
           newgrp 有效组的组名
           注意：newgrp命令会启动一个子进程；新创建的文件(或目录)的所属组是有效组。

二：显示文件内容的命令
    1.  cat [选项] 文件名
        选项： -n 显示行号
        eg: cat -n apple.txt  # 显示apple.txt文件的内容，并加上行号
        
        
        cat还有重定向、追加内容的作用：
        cat > 文件名  # 在键盘输入内容，并将内容重定向（保存）到指定的文件中,ctrl+d退出

        cat 文件名1 [文件名2 ...] > 新文件名  # 将多个文件内容重定向到新文件中

        cat >> 文件名 # 从键盘输入内容，追加到指定文件中

        cat 文件名1 [文件名2 ...] >> 新文件名  # 将多个文件内容追加到新文件中

    2.  more [选项]
        more命令经常与其他命令结合使用，用来分屏显示内容，"Enter"显示下一行，
        空格键显示一屏。
        eg: find / | more -d

    3.  less [选项]
        less命令是more命令的加强版，可以使用方向键（上下键）查看内容

    4.  head [选项] 文件名
        作用：查看文件的前几行内容
        head -n 3 apple.txt  # 显示apple.txt文件中前三行内容
        简写形式： head -3 apple.txt   # 显示apple.txt文件中前三行内容       

        eg: head -n 3 apple.txt | cat -n > ok.txt

    5.  tail [选项] 文件名
        作用：查看文件的后几行内容
        tail -n 2 apple.txt  # 显示apple.txt文件中最后两行内容
        简写形式： tail -2 apple.txt 

三：vi(vim)编辑文件
    一般模式：vi(vim)进入文件时的模式，可以在此模式下使用快捷键删除、复制、
              粘贴、光标移动等操作。
    编辑模式（输入模式）：在一般模式中按a、i等键进入编辑模式。按Esc进入一般模式。
    命令行模式（末行模式）：在一般模式中按冒号:或者/或者?可以进入命令行模式。
     注意：命令行模式的冒号“:”后面可以输入命令: 
                                w(写入)、q(退出)、!、set number(设置行号)。
vi一般模式下的常用快捷键：
     x:删除光标位置的字符
     X:删除光标前面的字符
     dd:删除当前行
     D:删除从光标所在字符至行尾的内容
     $:将光标移动至当前行的末尾
     ^:将光标移动至当前行的开头
     G:光标跳转至最后一行的行首
     y$:从光标处开始复制到行尾
     p: 粘贴
